# Tailwind CSS Styling Skill

## Purpose
Guidelines for building fast, consistent, and responsive user interfaces with Tailwind CSS.

## Best Practices
- Enforce the utilization of theme variables (colors, spacing, typography) rather than arbitrary values (e.g. `w-[17px]`).
- Design layouts using mobile-first utility classes (`sm:`, `md:`, `lg:`).
- Keep HTML structures clean using semantic groupings.
